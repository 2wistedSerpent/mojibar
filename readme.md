## Usage

`ctrl + shift + space`<br>
Open app.

:arrow_up_small::arrow_down_small::arrow_backward::arrow_forward:<br>
Navigate between emojis.

`enter`<br>
(when focusing on an emoji) Copy emoji code and exit. For example: `:poop:`.

`shift + enter`<br>
(when focusing on an emoji) Copy emoji unicode char and exit. For example: `ðŸ’©`.

`/`<br>
Jump to the search field.

`escape`<br>
Exit.
