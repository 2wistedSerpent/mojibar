## Usage

`ctrl + shift + space`<br>
Open app.

`🔼🔽◀️▶️`<br>
Navigate between emojis.

`enter`<br>
(when focused on an emoji) Copy emoji code and exit. For example: `:poop:`.

`shift + enter`<br>
(when focused on an emoji) Copy emoji unicode char and exit. For example: `💩`.

`/`<br>
Jump to the search field.

`escape`<br>
Exit.
